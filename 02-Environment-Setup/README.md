# 02 — Environment Setup

Before any AD work happens, both VMs need to exist and be networked correctly. Getting this wrong early causes confusing DNS and domain-join issues later, so it's worth doing carefully the first time.

- [I. VirtualBox Network Setup](./I.%20VirtualBox-Network-Setup.md) — the DC's two NICs (NAT + Internal Network)
- [II. Client VM Setup](./II.%20Client-VM-Setup.md) — the domain-joined client, on the same internal network
